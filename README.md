# Walkerpedia

**星城亜美** が執筆しているオリジナル世界観「Walkerverse」の設定資料集です。  
Obsidianを用いて構築しており、現在は **20万字以上の設定資料** を収録しています。  
本文はまだ1万字程度ですが、膨大な裏設定を少しずつ形にしています。

---

## 📖 内容
- キャラクター資料
- 世界観（地理・歴史・組織）
- 魔法体系の設定
- 架空現代史の差分記録
